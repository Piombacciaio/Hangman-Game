# Hangman Game

Simple version of the classic hangman game

## Supported languages (From API)

- English
- Italian
- German
- Spanish
- Chinese

## Requirements

- [Python 3.x](https://www.python.org/downloads/) (Made with Python [3.9.4](https://www.python.org/downloads/release/python-394/))
